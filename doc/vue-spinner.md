## [vue-spinner](https://github.com/greyby/vue-spinner)

> 安装

```text
npm install --save vue-spinner

```

> 使用

```text
import { PulseLoader } from 'vue-spinner/dist/vue-spinner.min.js';

components: {
  PulseLoader,
}

```

```text
<li class="spinner-list">
    <ring-loader></ring-loader>
</li>
          
```
